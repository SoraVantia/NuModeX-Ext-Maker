### 3.4 Uso de Tokens y Costos de API

Cuando usted utiliza NuModeX Ext Maker para crear, editar, mejorar o conversar sobre extensiones y sitios web, el Software envía solicitudes al proveedor de IA que haya seleccionado utilizando su clave API. Cada solicitud incluye su entrada (texto del prompt, historial de conversación e imágenes adjuntas) así como instrucciones internas (system prompts) utilizadas por el Software para guiar el comportamiento y formato de salida del modelo de IA. Estas instrucciones internas son propiedad exclusiva de SoraVantia GK y no son visibles para el usuario, pero contribuyen al recuento total de tokens de cada solicitud de API.

Debe tener en cuenta los siguientes factores que afectan sus costos de API:

- Cada solicitud de API incluye el system prompt interno del Software, que consume tokens además de su propia entrada. El tamaño del system prompt varía según la operación (creación, edición, chat, mejora) pero puede oscilar entre varios cientos y varios miles de tokens por solicitud.
- El historial de conversación se incluye en las solicitudes posteriores dentro de la misma sesión, lo que significa que el uso de tokens aumenta a medida que la conversación se alarga.
- La creación o mejora de una extensión o sitio web generalmente consume más tokens que un simple mensaje de chat, porque el modelo de IA genera el contenido completo de los archivos en su respuesta.
- Los diferentes modelos de IA tienen diferentes precios por token. Los modelos más grandes y capaces generalmente cuestan más por token que los modelos más pequeños. Consulte la página de precios de su proveedor de IA para conocer las tarifas actuales.
- Algunos proveedores de IA ofrecen funciones como el almacenamiento en caché de prompts que pueden reducir los costos de solicitudes repetidas. La aplicación de estas funciones depende del proveedor y está fuera del control de SoraVantia GK.

SoraVantia GK no controla, establece ni recibe ninguna parte de las tarifas cobradas por los proveedores de IA. Todos los costos de API son facturados directamente por el proveedor de IA a usted según su consumo de tokens. SoraVantia GK no es responsable de costos de API inesperados o superiores a los previstos resultantes de su uso del Software, independientemente de la causa.
