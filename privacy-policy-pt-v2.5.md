## 2. O QUE É O NUMODEX EXT MAKER

O NuModeX Ext Maker é uma extensão de navegador Manifest V3 que utiliza IA para ajudar os utilizadores a criar extensões de navegador e sites estáticos. A extensão opera inteiramente dentro do seu navegador. Não há servidor backend, contas de utilizador, início de sessão, subscrição nem registo. Os utilizadores fornecem as suas próprias chaves API para serviços de IA.

---

## 3. DADOS QUE NÃO RECOLHEMOS

A SoraVantia GK não recolhe, recebe, armazena, processa nem transmite quaisquer dados pessoais através do NuModeX Ext Maker. Especificamente, a SoraVantia GK não recolhe:

- O seu nome, endereço de correio eletrónico nem qualquer informação de contacto
- O seu endereço IP nem identificadores de dispositivo
- O seu histórico de navegação nem atividade de navegação
- Os seus dados de localização
- As suas chaves API nem quaisquer credenciais
- Os seus prompts, histórico de conversa nem qualquer conteúdo que crie
- Quaisquer dados de utilização, dados comportamentais nem dados de interação
- Quaisquer cookies, impressões digitais nem identificadores de rastreamento

A SoraVantia GK não dispõe de servidores, bases de dados, plataformas de análise nem infraestrutura de rastreamento para o NuModeX Ext Maker. A extensão não contém código de análise, código de telemetria nem código de rastreamento de qualquer tipo.

---

## 4. DADOS ARMAZENADOS LOCALMENTE NO SEU NAVEGADOR

O NuModeX Ext Maker armazena os seguintes dados localmente no seu navegador. Estes dados nunca saem do seu dispositivo e nunca são enviados à SoraVantia GK:

- **Chave API:** A chave API do seu fornecedor de serviços de IA é armazenada no armazenamento local de extensões do navegador. É utilizada exclusivamente para autenticar pedidos ao fornecedor de IA que selecionou. A SoraVantia GK não pode aceder, visualizar nem recuperar a sua chave API.

- **Estado de aceitação do EULA:** Um registo de se aceitou o Contrato de Licença de Utilizador Final, armazenado no armazenamento local de extensões do navegador.

- **Preferências de interface:** As suas preferências de interface como idioma selecionado e definição de modo escuro, armazenadas no armazenamento local de extensões do navegador.

- **Modelo de IA selecionado:** O modelo de IA que escolheu para a geração, armazenado no armazenamento local de extensões do navegador.

- **Projetos:** Os seus ficheiros gerados, histórico de conversa e histórico de anulação são armazenados em IndexedDB dentro do seu navegador. Estes são os seus produtos de trabalho e permanecem inteiramente sob o seu controlo.

- **Configuração de modelo personalizado:** Se configurar um endpoint de servidor de IA personalizado, o URL do endpoint, o nome do modelo e a chave API opcional são armazenados no armazenamento local de extensões do navegador.

**Como eliminar os seus dados armazenados localmente:**

- Desinstale a extensão NuModeX Ext Maker do seu navegador. Isto remove todos os dados armazenados localmente.
- Utilize as opções de eliminação dentro da aplicação para eliminar seletivamente chaves API e projetos individuais sem desinstalar.

## 5. DADOS ENVIADOS A FORNECEDORES DE IA DE TERCEIROS

Quando utiliza o NuModeX Ext Maker para criar, editar, melhorar ou conversar sobre extensões e sites, a extensão envia pedidos diretamente do seu navegador ao fornecedor de IA que selecionou. A SoraVantia GK não atua como intermediária e não tem acesso a quaisquer dados transmitidos entre o seu navegador e o fornecedor de IA.

### 5.1 O que é enviado ao fornecedor de IA

Cada pedido ao fornecedor de IA inclui:

- O seu texto de prompt (as instruções que escreve)
- O histórico de conversa da sessão atual
- Quaisquer imagens que anexe ao seu prompt
- O system prompt interno do Software (instruções proprietárias que orientam o comportamento e formato de saída do modelo de IA - não são visíveis para si mas são incluídas em cada pedido)

### 5.2 Como estes dados são transmitidos

Os seus dados são transmitidos diretamente do seu navegador para os servidores do fornecedor de IA utilizando a sua própria chave API. A transmissão utiliza o endpoint padrão da API do fornecedor de IA e é encriptada via HTTPS. Os servidores da SoraVantia GK não estão envolvidos nesta transmissão em momento algum.

### 5.3 Fornecedores de IA suportados

O NuModeX Ext Maker suporta a integração com vários fornecedores de IA, incluindo, sem limitação:

- Google (Gemini)
- OpenAI (GPT)
- Anthropic (Claude)
- Endpoints de servidor de IA personalizados configurados pelo utilizador
- Modelos de IA no dispositivo fornecidos pelo navegador

### 5.4 Políticas de privacidade dos fornecedores de IA

Cada fornecedor de IA tem a sua própria política de privacidade que rege a forma como trata os dados recebidos através de pedidos de API. A SoraVantia GK não tem controlo sobre a forma como os fornecedores de IA processam, armazenam ou utilizam os dados que lhes envia. É responsável por rever e compreender as políticas de privacidade dos fornecedores de IA que escolher utilizar.

### 5.5 Modelos de IA no dispositivo

Se utilizar modelos de IA no dispositivo fornecidos pelo seu navegador, o NuModeX Ext Maker não transmite os seus prompts para qualquer servidor externo. No entanto, a SoraVantia GK não controla a implementação dos modelos de IA no dispositivo pelo fornecedor do navegador. O fornecedor do navegador pode recolher telemetria, dados de utilização ou outras informações relacionadas com a utilização de modelos no dispositivo ao abrigo das suas próprias práticas de dados. A SoraVantia GK não tem conhecimento, controlo nem responsabilidade sobre qualquer recolha de dados efetuada pelo fornecedor do navegador em ligação com os modelos de IA no dispositivo. Deve consultar a política de privacidade do fornecedor do seu navegador para obter detalhes sobre como as funcionalidades de IA no dispositivo tratam os seus dados.

### 5.6 Endpoints de servidor de IA personalizados

Se configurar um endpoint de servidor de IA personalizado, os seus prompts são enviados diretamente para o endereço do servidor que configurou. A SoraVantia GK não tem conhecimento, controlo nem responsabilidade sobre endpoints personalizados. É o único responsável por compreender as práticas de dados, a segurança e as políticas de privacidade de qualquer servidor personalizado que configure.

## 6. SYSTEM PROMPTS E UTILIZAÇÃO DE TOKENS

O NuModeX Ext Maker inclui system prompts internos que são enviados com cada pedido de API para orientar o comportamento e formato de saída do modelo de IA. Estes system prompts são propriedade exclusiva da SoraVantia GK e não são visíveis para o utilizador. Não contêm quaisquer dados pessoais sobre si.

Os system prompts, juntamente com a sua entrada e o histórico de conversa, contribuem para a contagem total de tokens de cada pedido de API. Isto afeta os seus custos de API, que são faturados diretamente pelo fornecedor de IA. Para detalhes completos sobre utilização de tokens e custos, consulte a Secção 3.4 do Contrato de Licença de Utilizador Final.

## 11. SEGURANÇA

### 11.1 Segurança dos dados locais

Todos os dados armazenados pelo NuModeX Ext Maker são armazenados localmente no seu navegador utilizando a API de armazenamento de extensões do navegador e IndexedDB. A segurança destes dados depende da segurança do seu navegador e do seu dispositivo. A SoraVantia GK não tem acesso aos seus dados armazenados localmente e não pode protegê-los de ameaças no seu dispositivo.

### 11.2 Segurança da chave API

A sua chave API é armazenada localmente no armazenamento de extensões do navegador e é utilizada apenas para autenticar pedidos ao seu fornecedor de IA selecionado. A SoraVantia GK não tem acesso à sua chave API. É responsável por manter a sua chave API segura e não a deve partilhar com outros.

### 11.3 Dados em trânsito

Quando os prompts são enviados a fornecedores de IA de terceiros, são transmitidos via HTTPS diretamente do seu navegador ao fornecedor de IA. A infraestrutura da SoraVantia GK não está envolvida nesta transmissão. A segurança dos dados em trânsito para os fornecedores de IA depende das medidas de segurança do fornecedor de IA.

### 11.4 Segurança do código gerado

O NuModeX Ext Maker gera código de extensões de navegador e sites utilizando IA. A SoraVantia GK não revê, audita nem valida a segurança do código gerado. O código gerado pode conter vulnerabilidades de segurança. É o único responsável por rever a segurança de qualquer código gerado pelo Software antes da instalação, implementação ou distribuição.
