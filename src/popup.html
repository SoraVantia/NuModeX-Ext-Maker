🚀 How to Use NuModeX Ext Maker

Step 1: Get API Key

Click Settings (⚙️) → Get API key from the cloud provider → Paste and "Save Key"

Step 2: Select AI Model

Choose a model from the dropdown menu

Step 3: Accept Terms

Read and accept the Terms of Service

Step 4: Describe Extension

Type what you want to build in the chat

Example: "Create a password generator extension"
Step 5: Build Extension

Click "Build Extension" button and wait

Step 6: Download Files

Click "Download All as ZIP"

Step 7: Install in Chrome

Extract ZIP → Chrome Extensions → Developer mode ON → "Load unpacked"

Done! Your extension is now installed.
🌐 Build Websites

You can also build static websites. Describe what you want in the chat, then click "Build Website" instead of "Build Extension".

📂 Import Files

Import existing extension or website files to edit with AI. Click More (▾) → Import Files. Select one or multiple files. Subsequent imports merge into the current project.

🔑 API Keys Per Provider

Each cloud provider uses its own API key. When you switch models, the correct key loads automatically. Save a key, switch providers, save another — each is remembered separately.

💡 Tips for Best Results

Start simple and build up. Describe the core feature first, then use Edit and Improve to add more. Use a model with a larger context window for complex projects. If generation fails, simplify the prompt or switch to a larger model.
