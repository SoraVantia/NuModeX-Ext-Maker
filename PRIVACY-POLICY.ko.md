## 1. 소개

본 개인정보 보호정책은 SoraVantia GK("SoraVantia GK", "당사", "우리")가 개발한 NuModeX Ext Maker("서비스", "확장 프로그램" 또는 "소프트웨어")가 사용자의 정보를 어떻게 처리하는지 설명합니다. 당사는 사용자의 개인정보 보호에 최선을 다하고 있습니다.

NuModeX Ext Maker는 브라우저에서 완전히 실행되는 브라우저 확장 프로그램입니다. 이 제품을 위한 서버, 데이터베이스 또는 클라우드 인프라를 운영하지 않습니다. 사용자의 데이터를 수집, 수신, 저장하거나 접근하지 않습니다.

## 2. 수집하는 정보

**데이터를 수집하지 않습니다.** SoraVantia GK는 NuModeX Ext Maker 사용자로부터 개인 정보나 사용 데이터를 수집, 수신, 저장 또는 처리하지 않습니다. 백엔드 서버, 분석, 원격 측정 또는 어떤 종류의 추적도 없습니다.

## 3. 기기에 로컬로 저장되는 정보

NuModeX Ext Maker는 표준 브라우저 스토리지 API를 사용하여 브라우저에 데이터를 로컬로 저장합니다. 이 데이터는 기기를 떠나지 않으며 SoraVantia GK 또는 당사 관리 하의 제3자에게 전송되지 않습니다.

### 3.1 Chrome Storage API (chrome.storage.local)

Chrome의 내장 스토리지 API를 사용하여 다음 데이터가 저장됩니다:

- **API 키** — 사용자가 입력한 AI 제공업체 API 키. 로컬에 저장되며 선택한 AI 제공업체에 대한 요청 인증에만 사용됩니다.
- **EULA 수락 상태** — 서비스 약관에 동의했는지 여부 및 버전 정보.
- **UI 환경설정** — 선택한 언어 및 다크 모드 설정.
- **선택된 AI 모델** — 드롭다운에서 마지막으로 선택한 AI 모델.
- **사용자 정의 모델 구성** — 구성된 경우, 사용자 정의 AI 서버 엔드포인트 URL, 모델 이름 및 선택적 API 키.

### 3.2 IndexedDB (Dexie.js를 통해)

브라우저 로컬 데이터베이스인 IndexedDB를 사용하여 다음 데이터가 저장됩니다:

- **프로젝트** — 프로젝트 이름, 생성된 확장 프로그램 파일(소스 코드), AI와의 대화 기록 및 타임스탬프를 포함한 저장된 프로젝트.
- **실행 취소 기록** — 프로젝트별로 저장되는 실행 취소 기능을 위한 이전 파일 상태.

### 3.3 로컬 저장 데이터 삭제 방법

다음 방법으로 언제든지 로컬에 저장된 모든 데이터를 삭제할 수 있습니다:

- 확장 프로그램 제거(모든 데이터가 자동으로 삭제됨)
- 브라우저 설정을 통해 브라우저의 확장 프로그램 데이터 삭제
- 확장 프로그램의 "키 삭제" 버튼을 사용하여 저장된 API 키 삭제
- 프로젝트 선택기의 "삭제" 버튼을 사용하여 개별 프로젝트 삭제
- "처음부터 다시 시작" 버튼을 사용하여 현재 프로젝트 내용 삭제

## 4. 제3자에게 전송되는 정보

### 4.1 AI API 제공업체

NuModeX Ext Maker를 사용하여 브라우저 확장 프로그램을 생성, 편집하거나 채팅할 때, 프롬프트 텍스트와 대화 기록은 브라우저에서 선택한 AI 제공업체로 직접 전송됩니다. 이 전송은 사용자의 API 키를 사용하여 AI 제공업체의 API를 통해 이루어집니다.

프롬프트 데이터를 수신할 수 있는 AI 제공업체는 다음과 같습니다:

- **Google** (Gemini API) — Google의 API 서비스 약관 및 개인정보 보호정책에 따름
- **OpenAI** (GPT 모델) — OpenAI의 API 이용약관 및 개인정보 보호정책에 따름
- **Anthropic** (Claude 모델) — Anthropic의 API 서비스 약관 및 개인정보 보호정책에 따름

**중요:**

- SoraVantia GK는 이러한 전송에서 중개자 역할을 하지 않습니다. 브라우저가 AI 제공업체의 서버와 직접 통신합니다.
- SoraVantia GK는 사용자가 AI 제공업체에 보내는 데이터에 접근, 기록 또는 모니터링하지 않습니다.
- API 키는 인증을 위해 AI 제공업체에 직접 전송됩니다. SoraVantia GK는 사용자의 API 키를 수신하거나 볼 수 없습니다.
- AI 제공업체에 전송되는 데이터에는 텍스트 프롬프트, 대화 기록 및 첨부 이미지가 포함됩니다. API 키 저장, UI 환경설정 또는 기타 로컬 설정은 포함되지 않습니다.
- 제3자 AI 서비스 사용은 해당 개인정보 보호정책 및 서비스 약관에 의해 규율됩니다. 해당 정책을 검토하시기 바랍니다.

### 4.2 온디바이스 AI 모델

온디바이스 AI 모델(Chrome의 Gemini Nano 또는 Edge의 Phi-4 Mini 등)을 사용하는 경우, 프롬프트는 기기에서 완전히 처리됩니다. 온디바이스 모델 사용 시 외부 서버로 데이터가 전송되지 않습니다.

### 4.3 사용자 정의 / 로컬 AI 서버

사용자 정의 AI 서버 엔드포인트(사용자의 머신이나 로컬 네트워크에서 실행되는 로컬 호스트 서버 등)를 구성할 수 있습니다. 사용자 정의 엔드포인트를 사용할 때:

- 프롬프트와 대화 기록은 구성한 서버 주소로 직접 전송됩니다.
- SoraVantia GK는 사용자 정의 엔드포인트에 대해 인지, 통제 또는 책임을 지지 않습니다.
- 로컬 서버(예: localhost에서 실행)를 구성한 경우, 데이터는 완전히 사용자의 머신에 남습니다.
- 원격 서버를 구성한 경우, 데이터는 해당 서버로 전송됩니다. 연결하는 사용자 정의 서버의 개인정보 보호 관행을 이해하는 것은 사용자의 책임입니다.
- 사용자 정의 엔드포인트 URL, 모델 이름, 선택적 API 키는 chrome.storage.local을 사용하여 브라우저에 로컬로 저장됩니다.

### 4.4 기타 제3자 전송 없음

NuModeX Ext Maker는 다음에 데이터를 전송하지 않습니다:

- SoraVantia GK 서버(이 제품을 위한 서버가 없습니다)
- 분석 서비스(사용하지 않습니다)
- 광고 네트워크(사용하지 않습니다)
- 소셜 미디어 플랫폼
- 섹션 4.1 및 4.3에 나열되지 않은 기타 제3자
