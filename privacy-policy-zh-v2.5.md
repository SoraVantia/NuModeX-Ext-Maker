SoraVantia GK 不通过 NuModeX Ext Maker 收集、接收、存储、处理或传输任何个人数据。具体而言，SoraVantia GK 不收集：

SoraVantia GK 没有为 NuModeX Ext Maker 设置服务器、数据库、分析平台或跟踪基础设施。本扩展程序不包含任何种类的分析代码、遥测代码或跟踪代码。

## 4. 在您的浏览器中本地存储的数据

NuModeX Ext Maker 将以下数据本地存储在您的浏览器中。这些数据永远不会离开您的设备，也永远不会发送给 SoraVantia GK：

- **API 密钥：** 您的 AI 服务提供商 API 密钥存储在浏览器的本地扩展程序存储中。它仅用于向您选择的 AI 提供商验证请求。SoraVantia GK 无法访问、查看或检索您的 API 密钥。

- **EULA 接受状态：** 您是否已接受最终用户许可协议的记录，存储在浏览器的本地扩展程序存储中。

- **界面偏好：** 您的界面偏好（如所选语言和深色模式设置），存储在浏览器的本地扩展程序存储中。

- **选择的 AI 模型：** 您为生成选择的 AI 模型，存储在浏览器的本地扩展程序存储中。

- **项目：** 您生成的文件、对话历史和撤消历史存储在浏览器内的 IndexedDB 中。这些是您的工作成果，完全在您的控制之下。

- **自定义模型配置：** 如果您配置了自定义 AI 服务器端点，端点 URL、模型名称和可选的 API 密钥将存储在浏览器的本地扩展程序存储中。

**如何删除您的本地存储数据：**

- 从浏览器中卸载 NuModeX Ext Maker 扩展程序。这将删除所有本地存储的数据。
- 使用应用内的删除选项，在不卸载的情况下选择性地删除 API 密钥和单个项目。

## 5. 发送给第三方 AI 提供商的数据

当您使用 NuModeX Ext Maker 构建、编辑、改进扩展程序和网站或就其进行对话时，本扩展程序直接从您的浏览器向您选择的 AI 提供商发送请求。SoraVantia GK 不充当中介，也无法访问您的浏览器与 AI 提供商之间传输的任何数据。

### 5.1 发送给 AI 提供商的内容

发送给 AI 提供商的每个请求包括：

- 您的提示文本（您输入的指令）
- 当前会话的对话历史
- 您附加到提示的任何图片
- 本软件的内部系统提示（指导 AI 模型行为和输出格式的专有指令 - 您看不到这些指令，但它们包含在每个请求中）

### 5.2 数据如何传输

您的数据使用您自己的 API 密钥直接从浏览器传输到 AI 提供商的服务器。传输使用 AI 提供商的标准 API 端点，并通过 HTTPS 加密。SoraVantia GK 的服务器在任何时候都不参与此传输。

### 5.3 支持的 AI 提供商

NuModeX Ext Maker 支持与各种 AI 提供商的集成，包括但不限于：

- Google（Gemini）
- OpenAI（GPT）
- Anthropic（Claude）
- 用户配置的自定义 AI 服务器端点
- 浏览器提供的设备端 AI 模型

### 5.4 AI 提供商的隐私政策

每个 AI 提供商都有自己的隐私政策，规定其如何处理通过 API 请求接收的数据。SoraVantia GK 无法控制 AI 提供商如何处理、存储或使用您发送给他们的数据。您有责任审查并理解您选择使用的 AI 提供商的隐私政策。

### 5.5 设备端 AI 模型

如果您使用浏览器提供的设备端 AI 模型，NuModeX Ext Maker 不会将您的提示传输到任何外部服务器。但是，SoraVantia GK 不控制浏览器供应商对设备端 AI 模型的实现。浏览器供应商可能会根据其自身的数据实践收集与设备端模型使用相关的遥测、使用数据或其他信息。SoraVantia GK 对浏览器供应商因设备端 AI 模型而进行的任何数据收集不知情、不控制，也不承担责任。请查阅您的浏览器供应商的隐私政策，了解设备端 AI 功能如何处理您的数据。

### 5.6 自定义 AI 服务器端点

如果您配置了自定义 AI 服务器端点，您的提示将直接发送到您配置的服务器地址。SoraVantia GK 对自定义端点不知情、不控制，也不承担责任。您全权负责了解您配置的任何自定义服务器的数据实践、安全性和隐私政策。

NuModeX Ext Maker 包含内部系统提示，这些提示随每个 API 请求一起发送，以指导 AI 模型的行为和输出格式。这些系统提示是 SoraVantia GK 的专有信息，用户不可见。它们不包含关于您的任何个人数据。
