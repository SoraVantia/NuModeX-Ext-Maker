### 3.4 Token Usage and API Costs

When you use NuModeX Ext Maker to build, edit, improve, or chat about extensions and websites, the Software sends requests to the AI provider you selected using your API key. Each request includes your input (prompt text, conversation history, and any attached images) as well as internal instructions (system prompts) used by the Software to guide the AI model's behavior and output format. These internal instructions are proprietary to SoraVantia GK and are not user-visible, but they contribute to the total token count of each API request.

You should be aware of the following factors that affect your API costs:

- Every API request includes the Software's internal system prompt, which consumes tokens in addition to your own input. The size of the system prompt varies depending on the operation (building, editing, chatting, improving) but may range from several hundred to several thousand tokens per request.
- Conversation history is included in subsequent requests within the same session, which means token usage increases as the conversation grows longer.
- Building or improving an extension or website typically consumes more tokens than a simple chat message, because the AI model generates complete file contents in its response.
- Different AI models have different token pricing. Larger, more capable models generally cost more per token than smaller models. Consult your AI provider's pricing page for current rates.
- Some AI providers offer features such as prompt caching that may reduce costs for repeated requests. Whether these features apply depends on the provider and is outside the control of SoraVantia GK.

SoraVantia GK does not control, set, or receive any portion of the fees charged by AI providers. All API costs are billed directly by the AI provider to you based on your token consumption. SoraVantia GK is not responsible for unexpected or higher-than-anticipated API costs resulting from your use of the Software, regardless of the cause.
